# TrandsAI 定价体系方案

> 版本：v1.0 | 日期：2026-04-27 | 适用：AI 伴侣/情感陪伴产品

---

## 一、商业假设

| 指标 | 数值 | 说明 |
|------|------|------|
| 获客成本（CAC） | $4.00 / 用户 | 广告投放、渠道分成等综合成本 |
| 当前付费率 | 5% | 行业平均水平，目标优化至 8-10% |
| 用户生命周期（LT） | 6 个月 | 情感类产品留存较好的预估 |
| 目标 LTV/CAC | ≥ 3:1 | 健康 SaaS/消费品标准 |

**目标**：月费定价需覆盖获客成本 + 运营成本后仍有合理毛利。

---

## 二、Token 成本分析

### 2.1 单次对话成本

当前架构每次用户消息触发 **6-7 次 LLM 调用**（LangGraph 工作流）：

| 节点 | 输入 Token | 输出 Token | 功能 |
|------|-----------|-----------|------|
| think_node | ~2,500 | ~500 | 思考分析 |
| reflect_node | ~1,800 | ~200 | 反思+情绪更新 |
| creative_node | ~1,200 | ~300 | 创意脑暴 |
| respond_node | ~3,500 | ~300 | 生成最终回复 |
| extract_facts | ~400 | ~100 | 提取事实 |
| update_summary | ~400 | ~50 | 更新关系摘要 |
| **合计/次** | **~9,800** | **~1,450** | **~11,250 tokens** |

> 注：persona_evolve（人格进化）每 5 轮触发一次，已均摊计入。

### 2.2 模型单价对比

| 模型 | 输入单价 | 输出单价 | 单次对话成本 | 月均成本（30轮/天） |
|------|---------|---------|-------------|-------------------|
| Claude Sonnet 4 | $3.00 / MTok | $15.00 / MTok | ~$0.051 | **~$45.9** |
| DeepSeek V3 | $0.27 / MTok | $1.10 / MTok | ~$0.004 | **~$3.6** |
| GPT-4o | $2.50 / MTok | $10.00 / MTok | ~$0.039 | **~$35.1** |

**结论**：
- Claude 作为主力模型，$12.99 月卡**无法覆盖成本**（毛利为负）
- DeepSeek 作为主力模型，毛利率可达 **70%+**
- **策略**：DeepSeek 做默认，Claude 作为增值服务额外收费

### 2.3 其他成本项

| 项目 | 单价 | 月均 |
|------|------|------|
| Embedding（记忆检索） | ~$0.001 / 次 | ~$0.9 |
| 头像生成（SD API） | ~$0.02 / 张 | ~$0.6 |
| 朋友圈内容生成 | ~$0.01 / 条 | ~$0.3 |
| 服务器+数据库 | - | ~$0.5 / 用户 |

## 三、定价策略

### 3.1 核心原则

1. **免费层控成本**：严格额度 + 轻量模型，靠广告和转化漏斗盈利
2. **付费层保毛利**：DeepSeek 默认，Claude 作为 +$4.99 高级选项
3. **降低首付门槛**：周卡 $2.99 验证付费意愿，再推月卡/年卡
4. **情感溢价**：AI 伴侣是情感消费品，用户对价格敏感度低于工具类产品

### 3.2 定价表

#### 订阅层级

| 层级 | 价格 | 日额度 | 伴侣数量 | 模型 | 广告 |
|------|------|--------|---------|------|------|
| **免费版** | $0 | 10 条 | 1 个 | DeepSeek | 有 |
| **体验周卡** | $2.99 / 周 | 60 条 | 1 个 | DeepSeek | 无 |
| **月度会员** | $12.99 / 月 | 300 条 | 3 个 | DeepSeek | 无 |
| **年度会员** | $59.99 / 年 | 500 条 | 5 个 | DeepSeek | 无 |
| **终身会员** | $149.99 | 无限 | 无限 | DeepSeek | 无 |

> 注：300 条/天为软上限，防止 API 被刷，正常用户极少触及。

#### 增值服务

| 项目 | 价格 | 说明 |
|------|------|------|
| **Claude 高级模型包** | +$4.99 / 月 | 切换 Claude Sonnet，情感表达更细腻 |
| **额外伴侣位** | $0.99 / 个 | 超出套餐后的增量购买 |
| **头像风格包** | $1.99 / 包 | 古风 / 二次元 / 写实等主题风格 |
| **特殊剧情场景** | $0.99 ~ $2.99 | 情人节 / 生日 / 旅行等限定互动 |
| **去广告（免费用户）** | $1.99 / 月 | 移除广告，保留额度限制 |

#### 广告变现（免费层）

| 形式 | 收益预估 |
|------|---------|
| 开屏广告 | ~$0.3 / 千次 |
| 激励视频（看广告 +5 条额度） | ~$0.05 / 次 |
| 信息流广告 | ~$0.2 / 千次 |
| **免费用户月均广告收入** | **~$0.4 ~ $0.6** |

## 四、盈利模型测算

### 4.1 场景 A：保守（付费率 5%，未优化）

以每 1,000 新增用户为周期：

| 项目 | 计算 | 金额 |
|------|------|------|
| 获客成本 | 1,000 × $4.00 | -$4,000 |
| 付费用户 | 1,000 × 5% = 50 人 | - |
| 月卡收入 | 35 人 × $12.99 | +$454.65 |
| 周卡收入 | 10 人 × $2.99 × 4 周 | +$119.60 |
| 年卡/终身分摊 | 5 人 × ~$20/月 | +$100.00 |
| 免费用户广告 | 950 人 × $0.50 | +$475.00 |
| **月总营收** | - | **~$1,149** |
| 运营成本 | 50 付费 × $4 + 950 免费 × $0.8 | -$960 |
| **月毛利** | - | **~$189** |
| 获客回本周期 | $4,000 ÷ $189 | **~21 个月** |

> 回本周期过长，不健康。需优化付费率或提价。

### 4.2 场景 B：目标（付费率 10%，优化后）

| 项目 | 计算 | 金额 |
|------|------|------|
| 获客成本 | 1,000 × $4.00 | -$4,000 |
| 付费用户 | 1,000 × 10% = 100 人 | - |
| 月卡收入 | 60 人 × $12.99 | +$779.40 |
| 周卡→月卡转化 | 20 人 × $12.99 | +$259.80 |
| 年卡/终身分摊 | 15 人 × ~$25/月 | +$375.00 |
|  Claude 升级包 | 20 人 × $4.99 | +$99.80 |
| 免费用户广告 | 900 人 × $0.50 | +$450.00 |
| **月总营收** | - | **~$1,964** |
| 运营成本 | 100 付费 × $4.5 + 900 免费 × $0.8 | -$1,170 |
| **月毛利** | - | **~$794** |
| 获客回本周期 | $4,000 ÷ $794 | **~5 个月** |

> LTV ≈ $794 × 6 个月 = $4,764，LTV/CAC ≈ 1.2:1，仍偏低。
> 需进一步延长生命周期或提升 ARPU。

### 4.3 场景 C：健康（付费率 12%，ARPU 提升）

| 项目 | 计算 | 金额 |
|------|------|------|
| 付费用户 | 1,000 × 12% = 120 人 | - |
| 月均 ARPU | - | ~$18.5 |
| 月总营收 | 1,000 × $18.5 | **~$1,850** |
| 运营成本 | - | **~$1,200** |
| **月毛利** | - | **~$650** |
| LTV（12 个月） | $650 × 12 | **$7,800** |
| **LTV/CAC** | $7,800 / $4,000 | **1.95:1** |

> 接近健康线（2:1），再通过增值服务推动至 3:1。

## 五、关键优化方向

### 5.1 提升付费率（5% → 10%+）

| 措施 | 预期效果 |
|------|---------|
| 首充礼包（$0.99 解锁 3 天无限额度） | 降低首次付费心理门槛 |
| 亲密度系统绑定付费（高亲密度解锁专属对话） | 利用情感沉没成本 |
| 限时促销（首月月卡 $6.99） | 制造紧迫感 |
| 流失用户召回（7 天未登录送 3 天体验卡） | 二次激活 |

### 5.2 降低运营成本

| 措施 | 预估节省 | 优先级 |
|------|---------|--------|
| 合并 think + reflect 节点 | -20% token | P0 |
| 系统提示词缓存 | -30% 输入 token | P0 |
| 免费用户限制 max_tokens=1024 | -40% 成本 | P1 |
| 人格进化从每 5 轮改为每 10 轮 | -10% 长周期成本 | P1 |
| 头像接本地 SD / 低价 API | -80% 图片成本 | P2 |

### 5.3 提升 ARPU

| 措施 | 预期效果 |
|------|---------|
| 推出 "双人剧场" 等高级互动场景 | +$2.99/次 |
| 语音消息功能（接入 TTS） | 作为 $19.99 超级会员卖点 |
| 限定节日皮肤/装扮 | 季节限定的冲动消费 |
| 伴侣 "专属记忆" 导出（PDF 情书） | $0.99/次，情感溢价高 |

## 六、定价实施路线图

| 阶段 | 时间 | 动作 |
|------|------|------|
| **Phase 1** | 第 1-2 周 | 接入支付（Stripe/Paddle），上线额度限制系统 |
| **Phase 2** | 第 3-4 周 | 上线月卡/年卡，A/B 测试 $9.99 vs $12.99 |
| **Phase 3** | 第 2 月 | 上线周卡、首充礼包，监控付费漏斗转化 |
| **Phase 4** | 第 3 月 | 上线 Claude 升级包、头像风格包等增值项 |
| **Phase 5** | 第 4-6 月 | 根据数据调整定价，推终身会员和超级会员 |

## 七、关键监控指标

| 指标 | 目标值 | 监控频率 |
|------|--------|---------|
| 付费转化率 | ≥ 10% | 每日 |
| 月卡 ARPU | ≥ $12 | 每周 |
| 付费用户 7 日留存 | ≥ 60% | 每周 |
| 免费→付费转化周期 | ≤ 14 天 | 每月 |
| 单用户月均 Token 成本 | ≤ $4 | 每周 |
| LTV/CAC | ≥ 3:1 | 每季度 |

## 八、风险提示

1. **模型涨价风险**：DeepSeek 等国产模型若涨价，需快速切换备用模型
2. **付费率不达预期**：5% 是行业平均，情感类产品可能更低，需预留 3 个月测试期
3. **额度滥用**：300 条/天软上限需配合风控，防止脚本刷 API
4. **竞品价格战**：Character.AI 等免费产品可能挤压付费空间，需强化差异化（深度情感连接 vs 娱乐聊天）
